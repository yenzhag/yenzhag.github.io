MySQL

——-增

insert into 表名 [字段1，字段2..字段n]

values (值1，值2，值n);

——删

delete from 表名 （删除表内的所有内容，但是表还在）

where 限制条件

——改

update 表名 set 字段1=值1 字段2=值2…

where 更新条件

——查

select * from 表名 （查询表内所有内容）

排序查询

关键字为order by 与 asc,desc,通常位于表名之后
